## Section1 基因编辑
#### 1.1 概述
- **基因编辑**：一种新兴的、能精确对生物体基因组特定目的基因==进行修饰==的基因工程技术或过程。 #名词解释 
	- 通过基因编辑工具酶==在基因组特定位点产生DSB==，引发生物体NHEJ或HDR修复，由于该修复过程容易出错，会导致靶向突变
	- 2012年正式发现，2020年获得诺奖
	- 我国于==2023年4月==批准农作物基因编辑生产证书，在我国先获得证书的农作物是高油酸大豆，后续矮杆玉米、抗白粉病小麦都获得证书，总共10张证书
		- **三大主粮作物**：水稻、小麦、玉米 #重点 
	- 转基因技术：将外源或内源遗传物质随机插入宿主基因组，会引入外源基因(且若插入位点不好会破坏某些基因)，有基因逃逸风险（即插入基因转移到其他物种）
	- 基因编辑技术是在基因组特定位点对靶标基因进行①定点修饰，②不引入外源基因，但可能造成③基因脱靶👉公众接受度更高
	- 发展过程
		- ![[Pasted image 20260108210254.png|500]]
- CRISPR：Clustered Regularly Interspaced Short Palindromic Repeats（成簇规律间隔短回文重复） [[Chapter1 基因组编辑的发现及应用]]
	- 发展历程：
		1. 2012年Cas9对DNA靶向性的体外研究
		2. 2013年首次在真核细胞中验证Cas9基因编辑工程
		3. 2014年在植物中取得成功。且利用Cas9进行全基因组功能筛选，发现Apo-Cas9的晶体结构、Cas9与gRNA和靶DNA复合物的晶体结构
#### 1.2 基因编辑的原理与类别
> 原理：工具酶在基因组特定位置==产生位点特异性双链断裂DSB==，这种双链断裂可利用生物体内的非同源末端链接NHEJ(大部分情况)👉缺失/同源重组HDR👉插入，由于该修复过程容易出错，从而导致靶向突变
> ![[Pasted image 20260108210348.png|600]]
1.  **巨型核酸酶Meganuclease**
	- 本质是==脱氧核糖核酸内切酶==，难以找到天然存在的，需要人工合成，需要形成二聚体发挥作用
	- 能识别和切割大的DNA序列(14-40bp)
	- 缺点：没有向导容易乱切；耗费大量人力、物力、成本
2. **锌指核酸酶(ZFN,Zinc Finger Nucleases)**
	- ![[Pasted image 20251208134620.png|400]]
		- 锌指蛋白是一类具有DNA结合能力的蛋白质，蛋白内含有锌离子结合的结构域：锌指。每个锌指通常==能识别并与3到4个相邻的核苷酸碱基序列结合==，通过组合不同的锌指，可以根据需要精确地设计出能够特异性结合目标DNA序列的锌指蛋白
		- 由锌指蛋白结构域(ZFN)与FokI核酸内切酶的切割结构域组成👉更精确一些
		- 锌指识别并结合特定的DNA序列，FokI要二聚体才发挥作用，需要两个互补的ZFN分子与靶位点特异结合
	- 原理：将DNA双链剪开造成==双链缺口==，可以定向切割，在真正意义上实现基因编辑。![[Pasted image 20251208134716.png|400]]
	- 缺点：易脱靶，且会发生单个ZFN结合DNA后就发生切割的情况
3. **TALEN转录激活样效应因子核酸酶**
	- 一类特异性DNA结合蛋白，由转录激活因子效应因子TAL和核酸内切酶FokI的剪切区域融合而成。相当于转录因子，结合区域更准确，识别更具特异性
	- TAL结合，FokI切割
	- 缺点：会脱靶，模块组装过程繁琐

#### 1.3 操作流程
> ![[Pasted image 20251208142401.png|600]]
1. 根据核酸工具酶和目的基因的序列特征，设计并==合成靶序列引物==
	- 靶点设计数据库：
	- 脱靶检测
	- 引物设计与合成
2. 利用酶切连接方法，构建可利用于基因编辑的==分子载体==
3. 将基因编辑分子载体用RNP递送系统或农杆菌导入植物细胞
4. 经过植物组织培养，获得基因编辑再生苗
5. 经PCR或分子测序，筛选==鉴定阳性==的基因编辑植株
6. 根据子代表型获得目标性状的基因编辑新种质
- 实战操作：老师上课发的小纸条👉在寻找PAM序列时，应当同时考虑正义链与反义链
