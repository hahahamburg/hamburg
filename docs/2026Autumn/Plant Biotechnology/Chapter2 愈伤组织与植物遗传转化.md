## Section1 愈伤组织诱导培养
>![[Pasted image 20251203114500.png|400]]
#### 1.1 愈伤组织(callus) #考过 
- 定义：将植物上的某个部分(组织或器官)切下，形成外植体，接种到适宜的培养基上培养，其外植体组织伤⼝⾯形成==新生的细胞团==
	- 离体培养条件下植物外植体上长出的⼀团⽆序生长的细胞
- 诱导培养
	- 目的意义
		1. 植株大规模再⽣，作为遗传转化受体
		2. 体细胞无性变异somaclonal variants
		3. 药物、抗体等产⽣，次⽣代谢产物
		4. 悬浮细胞系、原⽣质体培养
#### 1.2 愈伤组织形成方法
- 理论依据
	1. **植物细胞的全能性**：植物的每个细胞在特定条件下都有再生成新个体的潜能👉受精卵>生殖细胞>体细胞
		- 实现条件：解除抑制性影响、给予刺激👉刺激信号Auxins、Cytokinin
			- ![[Pasted image 20251203113920.png|200]]
	2. **体细胞重编程(somatic reprogramming)**：分化的体细胞在特定的条件下==被逆转后恢复到全能性==状态，或者形成胚胎干细胞系，或进⼀步发育成⼀个新的个体的过程
		- 方法：核移植、细胞融合、细胞提取物诱导、化学诱导以及分⼦调控诱导等
	3. **脱分化**(dedifferentiation)：⼀个成熟细胞==转变为分生状态==的过程；或已分化的细胞，当受到创伤或进⾏离体培养(也受到创伤)时，已停⽌分裂的细胞⼜重新恢复分裂，细胞改变原有的分化状态，失去原有结构和功能，成为具有未分化特性的细胞的过程

	1. **非胚性愈伤组织**(non-embryonic callus,NEC)：细胞体积较⼤，==不易成团==，孔隙多
	2. **胚性愈伤组织**(EC)：质地较坚实，==颜色有乳白色或黄色==，表面具球形颗粒，其生长缓慢👉具有**全能性表达潜能**
		- 分为致密型/易碎型EC #考过 
		- 记忆特征：质地疏松、细胞排列稀疏、细胞质稀薄、核质比低
- 培养方法
	- **悬浮培养**(suspension culture)：: 把一些小块生长旺盛的愈伤组织放入液体培养基中，==进行振荡培养==，从而使愈伤组织块变成良好分散性的细胞和小的细胞聚集体
		- 用途
			- 提供一个==相对一致的细胞群体==（供生化研究、胚胎发生研究、胚的同步化研究等);[[Chapter7 微生物的生长]]
			- 细胞增殖速度快，适于进行大规模培养,==分离原生质体==的良好细胞来源;
			- 研究次生代谢、酶诱导、基因表达、抗生素的降解的良好实验体系;
			- 多数悬浮细胞缺乏叶绿体、色素，有利于分离酶和次生物质
	- **继代培养**(subculture)：指愈伤组织在培养基上生长一段时间后，营养物枯竭，水分散失，并已经积累了一些代谢产物，此时需要将这些组织转移到新的培养基上👉与我们的实验联系起来(每次都要转板)
#### 1.3 影响因素
1. 基因型(外植体)[[Chapter1 植物组织培养成功的关键因素]]
	- 基本原则：外植体的选择一般以幼嫩的组织或器官，==顶端分生组织==为宜
		- e.g.种子；在无菌条件下发芽种子产生的根、茎、叶(无菌苗)
	- 大小：外植体的组织块要达到2万个细胞(5-10mg)以上才容易成活
2. 培养基及激素配⽐
3. 培养条件：光照、温度、外植体预处理

#### 2.1 根癌农杆菌介导
- 系统分类
	1. ⼀元(共整合)载体系统→因为重组质粒同时包含目的基因+标记基因和Vir基因
		- ![[Pasted image 20251203115547.png|300]]
		- 包含2个独立的质粒：农杆菌质粒、⼤肠杆菌中间载体
		- 过程：中间载体与Ti质粒通过==同源重组使外源基因整合到T-DNA区==，共整合质粒在农杆菌中稳定存在
	2. **二元载体系统**👉现在常用 
		- ![[Pasted image 20251027145206.png|400]]
		- 2个独立的质粒：
			- 含有T-**DNA的中间载体**(微型Ti质粒)：含外源==目的基因==、25bp的左右边界序列、==筛选标记基因等，无Vir==→作为穿梭载体
			- **卸甲Ti质粒**：无T-DNA、有Vir→辅助载体，只用于切割目的基因
		- 当农杆菌侵染受伤的植物时，卸甲Ti质粒上的==Vir基因受植物的创伤信号启动==，与微型Ti质粒上的左右边界序列发⽣反式相互作⽤，将微型Ti质粒的T-DNA切割下来并转移进⼊植物基因组中
		- 优点：天然Ti质粒体积庞大，而二元表达载体体积较小；安全性较高(辅助质粒缺失T-DNA区，不会诱导植物产生冠瘿瘤)
	3. 三元载体系统👉适用于难转化植物(单子叶)和高效育种
		- 组成：
			- 双元载体：目的基因+标记基因
			- 制毒辅助载体：主要Vir区基因
			- 复制辅助载体：含有辅助Vir基因/增强子元件，提升目的基因的整合效率
		- ![[Pasted image 20251203115926.png|500]]
2. 菌株类型
	- 农杆菌染色体背景：主要有C58和Ach5 #考过 
		- ==C58背景==的菌株生长速度快、不结球，对双子叶植物侵染效率高
		- Ach5背景的菌株则生长慢、易结球，在转化实验中难以操作，培养后不易洗掉
	- **菌株侵染力**：农杆碱型(Succinamopine)>胭脂碱型(Nopaline)>章鱼碱型(Octopine) 指的是产生的冠瘿碱类型
	- 理解农杆菌染色体背景： #课后拓展 
		- Concepts：农杆菌**自身生存和代谢的遗传基础**，它不直接参与 T-DNA 的转移，但决定了农杆菌的生长性能和侵染植物的能力
		- 不同农杆菌菌株的染色体背景存在差异，会影响其作为转基因载体的效率
3. 基本原理： #待解决 
	- ![[Pasted image 20251203120537.png|600]]
	1) 农杆菌依附于植物表⾯伤⼝，植物分泌的小分子化合物诱导VirA和VirG等基因表达Vir产物
		- 单子叶植物（如小麦、水稻）受伤后分泌的酚类物质较少，因此在转基因实验中，常需在共培养基中**人工添加乙酰丁香酮**，以激活农杆菌的侵染能力
	2) 诱导Ti质粒产⽣⼀条新的T-DNA单链分⼦
	3) 单链分⼦从Ti质粒上脱离并与VirD2共价结合
	4) 与VirD4、VirB和VirE2等蛋⽩结合形成**T复合物(T-complex)**，进⼊植物细胞核并整合到基因组中
	5) 通过**VirB蛋白转运体**进入植物细胞内，类似于细菌的结合转移
		- T-DNA整合过程中的未解问题：
			- 整合到基因组时是单链/双链
			- 插入的基因组位点：通常插入在基因的5ʹ和3ʹ区域，而较少整合到着丝粒和端粒区域；优先整合在富含A-T的区域 (这些区域的DNA双螺旋稳定性相对较低)
			- ![[Pasted image 20251203121408.png|500]]
4. 操作过程：愈伤组织诱导培养（外植体准备）→外植体/农杆菌共培养→筛选培养→分化培养
	- ![[Pasted image 20251203121436.png|500]]
5. 主要影响因素
	1) 基因型：物种、外植体类型、外植体发育时期
		- e.g.拟南芥浸花法（开花后在菌液里泡一泡，后培养收种子即可）
		- 大多数作物都需要组织培养的方法
	3) 抑菌筛选过程：抑制农杆菌生长的抗生素👉一般用Timentin最有效、外植体褐化
	4) 农杆菌：菌株类型、农杆菌生长阶段、悬浮培养基、侵染液浓度、侵染液中是否添加诱导物
	5) 表达载体：载体类型、外源片段大小、筛选标记
		- **Vir基因活化诱导物**：酚类化合物、单糖、氨基酸、低pH 
		- 诱导物添加方法：
			- 在制备农杆菌侵染液4-6h前，或者加在侵染液中；
			- 在共培养培养基中添加
			- 在侵染液和共培养基中均添加。
	6) 侵染过程：外植体是否预处理、侵染方式方法及时长、侵染过程中是否超声或抽真空处理
	7) 共培养过程：共培养基、是否添加诱导物、共培养温度及时长

- 影响因素：微粒载体及可裂膜、颗粒大小、渗透处理、转化前后处理适当时间
- 优缺点：(现在不常用，但一些瞬时表达实验也会使用)
	- 优点：
		- 快速；
		- 农杆菌法无法转化大部分的单子叶植物，同时一般直接基因转移只能转化单子叶植物的原生质体；而基因枪法能够==成功转化单子叶植物的完整细胞==；
		- 可以将DNA传递到植物细胞基因组/叶绿体及线粒体基因组
		- 不依赖宿主基因型
	- 缺点：可能导致多重/混乱整合
#### 2.3 病毒诱导的植物遗传转化
- 方法：
	1. 植物体表面存在伤口，此时病毒外壳蛋白会与植物细胞膜上的特异性受体蛋白结合完成吸附
	2. 病毒靠着外壳蛋白与细胞膜融合/内吞作用进入植物细胞内释放核酸
- 案例：携带有PDS片段的病毒载体，利用农杆菌注射法接种到棉花子叶，植株不同发育时期的表型

- e.g.HIGS技术以病原菌生长发育和参与致病的重要基因作为靶点，寄主植物中表达与病原菌靶基因互补的RNA分子，再经植物细胞加工形成siRNA，当病原菌侵染寄主植物时，寄主植物能够将siRNA转运到病原菌细胞内，特异性识别病原菌靶基因的mRNA，导致其降解，从而干扰靶基因的转录和翻译，进而影响病原菌的正常生长发育而降低病害
