> **细胞分化**(cell differentiation)：同⼀来源的细胞逐渐产⽣出形态结构、功能特征各不相同的细胞类群的过程，其结果是在空间上细胞产⽣差异，在时间上同⼀细胞与其从前的状态有所不同
> 
> 细胞分化的本质：基因组在时空上的==选择性表达==
> 
> **细胞再分化**(cell redifferentiation)：已经去分化形成的胚性细胞在某些刺激条件下，重新启动开始分化发育的过程

#### 1.2 器官发生(organogenesis)
>指培养条件下的组织或细胞团（愈伤组织）分化形成不定芽/根等器官的过程
>![[Pasted image 20251222083607.png|400]]
- **直接器官发生**：器官由切下的外植体中已存在的==器官原基直接发育而成==，不经过愈伤组织阶段
	1) 外植体中已存在器官原基，通过组织培养形成相应组织器官并再生植株，如茎尖、根尖分⽣组织培养
	2) 外植体某些部位的细胞，在重新分裂后直接形成分生细胞团，然后由分生细胞团形成器官原基
- **间接器官发生**：外植体脱分化==形成愈伤组织==，由愈伤组织诱导产⽣分生细胞团，再分化成不同的器官原基，进而形成器官
	- 三个阶段：形成愈伤组织→形成分生中心→器官原基及器官形成(==脱分化→再分化==)
	- 三种形式：
		1) 先芽后根：愈伤组织上先生成芽，芽伸长后，在其基部直接生根。==最为普遍==
		2) 先根后芽：与上者相反。双子叶植物普遍，单子叶较少
		3) 根芽同时生长：只有维管束相连才能成活
	- 需要强脱分化激素(2,4-D)，再分化阶段需要严格调控激素比值
#### 1.3 胚胎发生(embryogenesis)
>离体培养下没有经过受精过程，==直接发育形成具有完整两极性（根尖和茎尖）的胚状结构==，这些结构在形态上与合子胚相似，并能进一步发育成完整的植株
![[Pasted image 20251222083555.png|400]]
1. **直接胚胎发生**：从培养的器官、组织、细胞或原⽣质体==直接分化形成胚==👉与直接器官发生有什么区别呢？ #待解决 
2. **间接胚胎发生**：外植体先愈伤化，再由愈伤组织细胞分化成胚。
- **胚状体**：起源于非合子细胞，经过胚胎发⽣和胚胎发育过程形成，是组织培养的产物
	- 按照来源分类：
		- **体细胞胚**：由普通植物体/孢子体的各种器官等双倍体的体细胞产⽣的胚状体
		- **花粉胚**：由小孢子或其分裂产物等单倍体细胞产⽣的胚状体，通常称为花粉胚，可发育成单倍体植株
	- 特点：两极性，即在其发⽣的最早阶段就具有根端（胚根）和茎端（胚芽）；它与⺟体植物或外植体的维管组织⽆解剖结构上的联系
- 优势：①数量多，②结构完整，③研究胚发育的良好材料体系→用于人工种子研究与生产，可以固定杂种优势
#### 1.4 两者比较及分子机理
>![[Pasted image 20251013151105.png|500]]
>器官发生的器官分化非同步，而胚胎发生的胚状体发育是同步的👉稳定性高
- 器官发生：**先长器官，再连成株**（芽和根是分开的，有维管连接）；
- 胚胎发生：**直接长胚，独立成株**（胚状体有完整胚结构，无维管连接）。
- 直接方式 = **外植体→器官 / 胚状体**（无愈伤组织）；
- 间接方式 = **外植体→愈伤组织→器官 / 胚状体**（有愈伤组织）。
## Section2 植物体细胞无性变异
>掌握：**定义**、性状表现及其**鉴定方法**
  **体细胞无性系变异**(Somaclonal variation)：泛指在植物细胞、组织或器官==离体培养过程==中，受到==非生物因子诱导==，进⽽导致培养细胞和再⽣植株发⽣==遗传变异或表观遗传变异==
  与传统变异的区别：变异频率高、类型丰富，可以定向筛选
#### 2.1 基本概念及性状表现
- 分类：
	- 变异来源：⾃发型变异/诱导型变异
	- 遗传特性角度：
		- 遗传变异：基因重组、基因突变和染⾊体变异
		- 表观遗传变异(可逆但不可稳定遗传)：DNA甲基化、组蛋白修饰、染⾊体重塑、转座⼦激活和基因沉默等
- 体细胞无性变异系的性状表现
	- 农艺性状的变异👉肉眼看
	- 代谢物成分及含量变异👉细胞/组织层面
	- DNA⽔平（染⾊体数⽬及结构变异、基因突变等）👉DNA分子水平层面![[Pasted image 20251013152825.png|400]]
#### 2.2 遗传基础 #重点 
- 细胞遗传学基础
	1. 染色体数⽬变异
	2. 染色体结构变异
- 分⼦遗传学基础
	1. 基因突变
	2. 基因的扩增、丢失或重排（核基因及细胞器基因）
	3. 表观遗传变化
	4. 组蛋⽩及小RNA的变化
	5. 转座⼦激活或失活
	6. 基因沉默👉部分表现为表观遗传

#### 2.4 应用
- 优缺点
	1. 优点
		- 诱变群体大、变异频率高、变化幅度大、筛选⽅便
		- 变异遗传稳定、后代稳定，诱变条件可控，不受季节限制，选育周期短，成本低等
		- 可在单细胞上进⾏，避免出现嵌合体
		- 保持原有优良遗传性状的前提下，仅改变个别农艺性状与其他物理化学诱变相结合，提⾼诱变效率
		- 可在培养基中添加特定选择压，获得特定突变体
		- 获得新的细胞质雄性不育系
		- 提供了植物基因变异的快捷途径，拓宽种质资源，加快育种进程
	2. 缺点
		1. 畸变率⾼
		2. 基因型依赖性
		3. 存在不稳定或不遗传的变异
- 研究
	1. 育种应用研究：⾼产优质、抗逆(⽣物和非生物逆境)、新的雄性不育种质
	2. 育种理论研究：利⽤变异系研究细胞培养、诱变中的遗传规律，其不受地理、环境、空间等限制
