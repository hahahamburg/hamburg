## Section1 分子标记辅助选择的原理
#### 1.1 MAS(Marker-Assisted Selection)
- Concepts：利用与目标性状紧密连锁的DNA分子标记对目标性状==进行间接或辅助选择==的现代育种技术
- **用于MAS的分子标记应具备3个条件**：
	- 分子标记与目标基因紧密连锁
	- 标记重复性好
	- 不同遗传背景和环境中选择有效
- **主要步骤**：选择目标基因/QTL→亲本选配(亲本多样性越大越好)→群体构建→多态性筛选→时代材料筛选 #重点 
- 准确率：基于检测标记与目标基因的距离来判断 #待解决 
	- ![[Pasted image 20251222151820.png|400]]
- 遗传基础
	1) 前景选择(foreground selection)：对==目标基因==的选择
		- ![[Pasted image 20251222152157.png|400]]
		- 选择的正确率会随重组率的增加而下降 #待解决 
		- ![[Pasted image 20251222152444.png|400]]
	2) 重组选择
	3) **背景选择**(background selection) 负选择：对==除了目标基因之外的基因组中其他部分==(即遗传背景)的选择
		- 选择的对象：几乎包括了整个基因组
		- 在分离群体中，由于在上一代形成配子时同源染色体之间会发生交换，因此每条染色体都可能是由双亲染色体重新组装的杂合体
		- ![[Pasted image 20251222152606.png|400]]
#### 1.2 MAS的类型
1. **数量性状MAS**：实际的成功例子很少，依赖于精确的遗传图谱
	- QTL定位越精细，MAS的可靠性越高，选择效果越好
2. **质量性状MAS**：存在表现型测定难度大、费用高且只能后期测量的问题
#### 1.3 分子标记辅助选择应具备的条件及优越性
- 条件
	1) 高效的分子标记基因型检测方法；
	2) 获得与目标基因共分离或紧密连锁的分子标记👉一般要求两者间的遗传距离小于5 cM,最好1 cM或更小
	3) 对于复杂的数量性状，遗传调控网络更好地发挥MAS的作用
- 优越性
	- 克服性状表型或基因型鉴定的困难；
	- 可以利用控制单一性状的多个（等位）基因，也可以同时对多个性状进行选择；
	- 能在任何时期进行选择，可在早期选择，提高选择强度；
	- 性状评价和选择不具有破坏性；
	- 可加快育种进程，提高育种效率，减轻连锁累赘程度
- 影响因素
	1) 分子标记与目的基因或QTL之间的连锁程度
	2) 性状的遗传率
	3) 群体大小：不小于200个
	4) 分子标记数量：有用标记越多越好
	5) 世代的影响
## Section2 MAS育种方法 
#### 2.1 回交育种
- 本质： **“借基因补短板”**：选一个综合性状都好，但唯独缺少某一个优良性状（比如抗病性）的品种作为 **轮回亲本**（相当于 “底子好的模板”）；再选一个携带目标优良基因的品种作为**供体亲本**（相当于 “基因供应商”）。通过杂交 + 多代回交，把供体亲本的**目标基因**导入轮回亲本，同时让后代的**绝大部分遗传背景恢复成轮回亲本的样子**—— 最终得到 “底子不变，短板补齐” 的新品种
	- 而 **MAS 的作用**，就是用分子标记精准追踪目标基因、加速背景恢复，比传统回交效率高得多
	- 核心：前景选择+背景选择
- ![[Pasted image 20251222153904.png|400]]
	- 最后一代回交个体自交，用标记筛选出**目标基因纯合**的个体，就是最终的 “高产抗病水稻”
#### 2.2 聚合育种
- 本质：本质是 **“攒基因叠 buff”**：把==分散在不同亲本中的多个优良基因==（比如抗病、抗逆、高产相关的多个基因），通过杂交和筛选，聚合到**同一个品种**中，让后代同时拥有多个优良性状
- 核心：多基因前景选择
- ![[Pasted image 20251222153914.png|400]]
#### 2.3 MAS选择的限制因素
- 作物基因作图研究进展缓慢，一些重要性状的分子标记与目标性状间的遗传距离仍较大，不符合MAS的要求
- 已筛选出的分子标记在不同遗传背景中表现不稳定，甚至同样的目标基因同样的标记在不同群体中其重组率会有差异。特别是呈数量性状遗传的性状的分子标记更易受遗传背景影响
- 利用分子标记技术进行MAS的成本较高，不能满足大规模育种选择的需要
- 生物技术手段与常规育种严重脱节
## Section3 MAS的应用
#### 3.1 基因/QTL转移
- 方法：多代回交结合前景选择(正选择)和背景选择(负选择)
	- 前景选择：保证每一回交世代选出的作为下一轮回交亲本的个体都包含目的基因
	- 背景选择
		- 加快遗传背景恢复成轮回亲本基因组的速度，以缩短育种年限；
		- 避免或减轻连锁累赘的问题。
- **基因转移**：在育种过程中==将有利基因/QTL==从供体亲本（地方品种、国外品种、野生种、远源种质）转移到受体品种的遗传背景中，以达到改良个别性状的目的
- 案例：
	1. 抗白叶枯基因的转移
	2. 华小黑1号的培育：水稻黑米新品种
		- ![[Pasted image 20251222182148.png|400]]
#### 3.2 基因/QTL聚合
- **基因聚合**(pyramiding)：将分散在几个不同品种的有利基因通过杂交方式聚合到一个基因组中，使某一性状表现更加突出的育种方法。
#### 3.3 分子设计育种
- **设计育种**：有目的地将重要农艺性状关键基因的==优异等位基因高效聚合==，设计并培育出人们所需要的新品种
	- e.g.1 基于SSSL文库的水稻设计育种平台
	- 实例2 高产优质水稻品种设计流程及品种培育
	- 实例3 异源四倍体野生稻快速从头驯化育种策略
